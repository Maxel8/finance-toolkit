# Finanz-Tool-Kit

Eine Sammlung von Finanzrechnern, geschrieben in Python.

## Funktionen

* Zinsrechner (einmaliges Kapital)
* Zinsrechner (ausschüttend)
* Zinseszinsrechner (thesaurierend)

## Installation

1. Repository klonen
2. Abhängigkeiten installieren

```bash
pip install -r requirements.txt
```

## Nutzung

```bash
python main.py
```

oder einzelne Rechner ausführen:

```bash
python interest.py
```
